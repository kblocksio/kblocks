# Command Line Tool

The Kblocks CLI provides several commands for working with blocks.

To install the CLI, follow the [Installation](../guide/installation.md) guide.

The CLI supports the following commands:

- [init](./init.md)
- [build](./build.md)
- [install](./install.md)
- [import](./import.md)
- [enrich](./enrich.md)
- [catalog](./catalog.md)
